
# Developer Portfolio with Terminal-Style Interactive Navigation

This project is a personal developer portfolio website built using **HTML, CSS, and JavaScript**.  
It is designed to showcase my profile, skills, projects, and contact details in a creative and professional way.

The unique feature of this portfolio is its **terminal-style interactive navigation**.  
Instead of using only normal buttons or menus, visitors can type commands like `about`, `skills`, `projects`, and `contact` to explore the portfolio.

---

## Project Objective

The goal of this project is to create a portfolio that is not only visually appealing but also interactive and memorable for recruiters, hiring managers, and employers.

This project demonstrates:
- frontend development skills
- creative UI design
- JavaScript-based interaction
- responsive design for desktop and mobile

---

## Main Features

- Personal hero section with introduction
- About section with background and story
- Skills section with technical skills
- Projects section with project cards
- Contact section with links
- Terminal-style command navigation
- Smooth scrolling between sections
- Responsive layout for mobile and desktop
- Clean and modern developer-style design

---

## Terminal Commands

The interactive terminal supports commands such as:

- `help` → shows available commands
- `about` → opens or scrolls to About section
- `skills` → opens or scrolls to Skills section
- `projects` → opens or scrolls to Projects section
- `contact` → opens or scrolls to Contact section
- `clear` → clears terminal output

This terminal feature is the main creative twist of the project.
